# BangB2 2026

Vietnamese driver license exam preparation app built with Swift and UIKit. Features 600 official theory questions, timed practice tests, detailed answer explanations, progress tracking, and offline study mode with encrypted local database.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Language | Swift |
| UI | UIKit, Storyboard |
| Database | Realm, SQLite (SQLCipher encrypted) |
| Networking | Alamofire, AlamofireObjectMapper |
| Navigation | ViewDeck (side menu) |
| CMS | Contentful |
| Storage | YapDatabase (SQLCipher) |
| Push | Firebase Core, Firebase Messaging |
| Ads | Google Mobile Ads (via hmdcommonkit) |
| In-App Purchase | StoreKit |
| Dependencies | CocoaPods |

## Key Features

- 600 official Vietnamese driving theory questions with detailed explanations
- Timed practice tests simulating real exam conditions
- Progress tracking with per-category statistics
- Encrypted local database (SQLCipher) for offline question storage
- Contentful CMS integration for dynamic content updates
- Side menu navigation with ViewDeck
- Firebase push notifications for study reminders
- StoreKit in-app purchases for premium content
- CSV data import for question bank updates
- App Intents for Siri and Shortcuts integration

## How It Works

1. **Data Loading**: Question bank data is stored in an encrypted SQLite database using SQLCipher. On first launch, questions are loaded and indexed by category (traffic signs, rules, situations).
2. **Study Mode**: Users browse questions by category, view detailed explanations, and mark questions for review. Progress is persisted in Realm.
3. **Practice Tests**: Timed exams are generated with randomized question selection matching official exam format. Results show per-question analysis with correct answers.
4. **Content Updates**: New questions and content can be fetched from Contentful CMS without app updates.
5. **Progress Tracking**: Study statistics (questions answered, accuracy rate, weak categories) are tracked and displayed on the dashboard.
